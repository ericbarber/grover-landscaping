# MVP Scope

## Primary Roles

- Company admin
- Crew manager
- Crew member
- Customer viewer

## MVP Features

### Admin

- Create customers
- Create properties
- Create crews
- Assign users to crews
- Create service jobs
- View completion reports

### Crew Member

- Log in from a mobile device
- View assigned jobs
- Open job details
- Capture before photos
- Capture after photos
- Add issue photos or notes
- Complete a checklist
- Mark job complete

### Customer or Manager

- View proof-of-completion report
- See timestamps
- See before/after photos
- See completion notes

## Out of Scope for MVP

- Billing and invoicing
- Route optimization
- Native mobile apps
- AI image validation
- Complex scheduling
- Customer self-service scheduling
- Multi-region deployment

## First Vertical Slice

```text
Login -> assigned job list -> job detail -> upload photo -> complete job -> view report
```
